# 基于 Socket.io electron  开发的一个即时聊天

## 代码演示

server
```
$ cd srever
$ npm install 
$ node start
```

client
```
$ cd client
$ npm install 
$ electron .
```


## 实现功能
- 登陆页面进入用户聊表页面，即时添加用户人数
- 点击列表内用户进入聊天页面,实现简单聊天功能


## 后续功能实现
- 单聊，群聊功能
- 完成文字消息,图片消息
- 文件传输
